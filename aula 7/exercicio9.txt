1. Qual é a diferença entre as declarações de variáveis var, let e const?
R: var é uma variável global, que não diferencia os escopos onde é declarada
let é uma variável global, que diferencia os escopos onde é declarada
const é uma variável global, que não altera seu valor

2. É verdade que o uso de parâmetros em funções é obrigatório?
R: Não é obrigatório
